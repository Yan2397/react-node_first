JavaScript մասնագիտացված կուրս

Հարգելի՛ ուսանող, front հատվածից՝ React, Angular կամ Vue համակարգերում
բոլոր տեսակի հարցումները ուղարկում եք http://localhost:5000 սերվերին

Ստորև ներկայացված են բոլոր հնարավոր հարցումները, որոնք կարելի է իրականացնել

հարցման տեսակ - GET կամ POST

հասցե - օրինակ՝ /login նշանակում է հարցումը պետք է գնա http://localhost:5000/login

մուտքային պարամետրեր - օրինակ {login:'ok'} նշանակում է պետք է ուղարկել login պարամետրեր

ելք օրինակ՝ {status:'ok'} նշանակում է ելքում կստանանք {status} օբյեկտը

Այսպիսով 

                
                axios.post('http://localhost:5000/login', {login:'inch vor mi ban'})
                    .then(r=>{
                        console.log(r.data);// {status...}
                    })

նշումներ տեսակ հասցե Նշանակություն մուտքային պարամետրեր ելք
POST /signup նախատեսված է գրանցվող մարդու տվյալները բազա ուղարկելու համար {name,surname, login, password} {status:'ok'} կամ {error:'message'}
POST /login նախատեսված է լոգին լինելու համար: {login, password} եթե նման մարդ չկա, ապա Error, կամ {token}
POST /getUserByToken Տրամադրում է User-ի տվյլաները՝ ըստ token -ի {token} {error:""} եթե նման մարդ չկա
user -ի տվյալները, եթե token -ը ակտիվ է և ճիշտ
POST /search Նախատեսված է մարդկանց որոնելու համար {token, text} users:[] որի մեջ կա՝ {name, surname, photo, ինչպես նաև areWeFriends, isRequestSent} դաշտերը
POST /getUserById վերադարձնում է User -ի տվյալներն ըստ Id -ի,լոգին եղած User -ի token {id,token} { mardu tvyalner@ {}}
POST /addFriend Նախատեսված է տվյալ մարդուն ընկերության հայտ ուղարկելու համար {լոգին եղած User -ի token, այն user -ի id-ին ում հայտ ենք ուղարկում} {token,id} {status:"ok"}
POST /getMyRequests Նախատեսված է լոգին եղած մարդուն ուղարկված ընկերության հայտերը ստանալու համար {token} {users:[]}
POST /getMyFriends Նախատեսված է լոգին եղած մարդու ընկերների ցուցակը ստանալու համար {token} {users:[]}
